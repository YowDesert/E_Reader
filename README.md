# E-Reader 漫畫＆PDF閱讀器 v3.0

一個功能豐富的JavaFX閱讀器應用程式，支援圖片、PDF檔案閱讀，以及OCR文字提取功能。

## 主要功能

### 📖 閱讀功能
- **多格式支援**: 支援 JPG、PNG、PDF 格式
- **雙模式顯示**: 圖片模式和文字模式無縫切換
- **智能翻頁**: 鍵盤、滑鼠、觸控多種翻頁方式
- **縮放功能**: 支援多種適應模式（適合寬度、高度、頁面、原始尺寸）

### 🎨 視覺體驗
- **多主題支援**: 淺色、深色、純黑、護眼、復古五種主題
- **響應式設計**: 自動適應不同螢幕尺寸
- **全螢幕模式**: 沉浸式閱讀體驗
- **專注模式**: 隱藏所有控制元件

### 📚 進階功能
- **書籤管理**: 支援添加、編輯、刪除書籤，可添加備註
- **OCR文字提取**: 從圖片和PDF中提取文字（需要Tesseract）
- **文字搜尋**: 在提取的文字中搜尋關鍵字並高亮顯示
- **閱讀統計**: 記錄閱讀時間和進度
- **自動翻頁**: 可自訂翻頁間隔的自動閱讀功能

### ⚙️ 個人化設定
- **護眼模式**: 降低藍光並定時提醒休息
- **夜間模式**: 自動根據時間切換主題
- **字體調整**: 文字模式下可調整字體大小和行距
- **記憶功能**: 記住上次閱讀位置和偏好設定

## 系統需求

- **Java**: JDK 17 或更高版本
- **JavaFX**: 17.0.2 或更高版本
- **記憶體**: 至少 512MB RAM
- **硬碟**: 至少 100MB 可用空間

### 可選依賴
- **Tesseract OCR**: 用於圖片文字識別功能
  - Windows: 下載安裝 Tesseract 並將 `tessdata` 資料夾放在應用程式目錄
  - 支援繁體中文和英文識別

## 安裝和運行

### 方法一：使用 Gradle 運行
```bash
# 確保已安裝 Java 17 或更高版本
java -version

# 確保已安裝 JavaFX
# 下載 JavaFX SDK 並設定環境變數
```

### 2. 下載和安裝
1. 下載 `E_Reader.jar` 檔案
2. 確保 `tessdata` 資料夾在正確位置
3. 確保 `LaTeX-OCR` 模組已正確安裝

### 3. 啟動應用程式
```bash
# 方法一：直接雙擊
雙擊 E_Reader.jar

# 方法二：命令行啟動
java -jar E_Reader.jar

# 方法三：指定 JavaFX 模組路徑
java --module-path "path/to/javafx-sdk/lib" --add-modules javafx.controls,javafx.fxml -jar E_Reader.jar
```

## 🚀 使用指南

### 基本操作
1. **啟動應用程式**: 雙擊 JAR 檔案
2. **選擇檔案**: 在檔案管理器中瀏覽並選擇檔案
3. **開始閱讀**: 使用滑鼠滾輪或方向鍵翻頁
4. **調整設定**: 點擊工具列按鈕調整各種設定

### 快捷鍵
- **方向鍵**: 上一頁/下一頁
- **Home/End**: 第一頁/最後一頁
- **Ctrl + F**: 搜尋功能
- **F11**: 全螢幕模式
- **Ctrl + +/-**: 縮放
- **R**: 旋轉圖片

### 進階功能
- **文字模式**: 點擊「📖 文字模式」按鈕
- **專注模式**: 點擊「🎯 專注」按鈕
- **自動閱讀**: 點擊「⏯ 自動閱讀」按鈕
- **書籤管理**: 點擊「🔖 書籤」按鈕

## ⚙️ 設定選項

### 主題設定
- **淺色模式**: 適合日間閱讀
- **深色模式**: 適合夜間閱讀
- **純黑模式**: 最大程度減少藍光
- **護眼模式**: 特殊色溫調整
- **復古模式**: 米色背景，減少疲勞

### OCR 設定
- **快速模型**: 快速識別，適合一般用途
- **最佳模型**: 高精度識別，適合重要文件

### 功能設定
- **自動儲存**: 自動儲存閱讀進度
- **記住最後檔案**: 啟動時自動載入上次檔案
- **觸控導航**: 支援觸控螢幕操作
- **顯示頁碼**: 顯示當前頁碼

## 🔧 故障排除

### 常見問題

#### 1. 應用程式無法啟動
**問題**: 啟動時出現錯誤
**解決方案**:
- 確認 Java 版本為 17 或更高
- 確認 JavaFX 已正確安裝
- 檢查系統記憶體是否充足

#### 2. OCR 功能無法使用
**問題**: 文字識別失敗
**解決方案**:
- 確認 `tessdata` 資料夾位置正確
- 檢查語言檔案是否完整
- 嘗試切換 OCR 模型

#### 3. 檔案載入失敗
**問題**: 無法開啟特定檔案
**解決方案**:
- 確認檔案格式是否支援
- 檢查檔案是否損壞
- 嘗試重新啟動應用程式

#### 4. 亮度調整異常
**問題**: 亮度調整效果不正確
**解決方案**:
- 使用預覽功能確認效果
- 點擊「應用設定」按鈕
- 重新啟動應用程式

## 📊 效能優化

- **圖片模式**: 約 200-500MB

- **PDF 載入**: 1-5 秒（視檔案大小）
- **EPUB 載入**: 2-8 秒
- **TXT 載入**: 1-3 秒
- **OCR 識別**: 10-30 秒/頁
