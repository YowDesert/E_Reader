# 📚 E_Reader - 智能電子書閱讀器

## 🎯 產品概述

E_Reader 是一款功能強大的智能電子書閱讀器，專為現代讀者設計。支援多種檔案格式，整合先進的 OCR 技術，提供舒適的閱讀體驗。

## ✨ 主要功能

### 📄 多格式支援
- **PDF 檔案**: 完整支援 PDF 閱讀，包含文字和圖片
- **EPUB 電子書**: 支援標準 EPUB 格式
- **TXT 文字檔案**: 新增支援純文字檔案
- **圖片檔案**: 支援 JPG、PNG、GIF、BMP、TIFF、WebP 等格式

### 👁️ 護眼功能
- **智能亮度調整**: 根據環境光線自動調整
- **護眼模式**: 減少藍光輻射，保護視力
- **夜間模式**: 自動切換，減少眼睛疲勞
- **亮度預覽**: 即時預覽亮度效果，確認後再應用

### 🔍 OCR 文字識別
- **Tesseract 5.5.1**: 高精度文字識別引擎
- **LaTeX OCR**: 數學公式識別
- **多語言支援**: 支援中文、英文等多種語言
- **快速/最佳模式**: 可選擇識別速度和精度

### 🎯 閱讀體驗
- **文字模式**: 純文字閱讀，支援字體大小調整
- **圖片模式**: 圖片瀏覽，支援縮放和旋轉
- **專注模式**: 隱藏干擾元素，專注閱讀
- **自動閱讀**: 自動翻頁功能
- **書籤管理**: 智能書籤系統

## 🛠️ 技術規格

### 系統需求
- **作業系統**: Windows 10 或更高版本
- **Java 版本**: Java 17 或更高版本
- **記憶體**: 建議 4GB 以上
- **磁碟空間**: 至少 2GB 可用空間

### 技術架構
- **開發語言**: Java 17
- **UI 框架**: JavaFX
- **OCR 引擎**: Tesseract 5.5.1
- **PDF 處理**: Apache PDFBox
- **EPUB 處理**: Epub4j
- **數學公式識別**: LaTeX OCR

## 📦 安裝說明

### 1. 環境準備
```bash
# 確保已安裝 Java 17 或更高版本
java -version

# 確保已安裝 JavaFX
# 下載 JavaFX SDK 並設定環境變數
```

### 2. 下載和安裝
1. 下載 `E_Reader.jar` 檔案
2. 確保 `tessdata` 資料夾在正確位置
3. 確保 `LaTeX-OCR` 模組已正確安裝

### 3. 啟動應用程式
```bash
# 方法一：直接雙擊
雙擊 E_Reader.jar

# 方法二：命令行啟動
java -jar E_Reader.jar

# 方法三：指定 JavaFX 模組路徑
java --module-path "path/to/javafx-sdk/lib" --add-modules javafx.controls,javafx.fxml -jar E_Reader.jar
```

## 🚀 使用指南

### 基本操作
1. **啟動應用程式**: 雙擊 JAR 檔案
2. **選擇檔案**: 在檔案管理器中瀏覽並選擇檔案
3. **開始閱讀**: 使用滑鼠滾輪或方向鍵翻頁
4. **調整設定**: 點擊工具列按鈕調整各種設定

### 快捷鍵
- **方向鍵**: 上一頁/下一頁
- **Home/End**: 第一頁/最後一頁
- **Ctrl + F**: 搜尋功能
- **F11**: 全螢幕模式
- **Ctrl + +/-**: 縮放
- **R**: 旋轉圖片

### 進階功能
- **文字模式**: 點擊「📖 文字模式」按鈕
- **專注模式**: 點擊「🎯 專注」按鈕
- **自動閱讀**: 點擊「⏯ 自動閱讀」按鈕
- **書籤管理**: 點擊「🔖 書籤」按鈕

## ⚙️ 設定選項

### 主題設定
- **淺色模式**: 適合日間閱讀
- **深色模式**: 適合夜間閱讀
- **純黑模式**: 最大程度減少藍光
- **護眼模式**: 特殊色溫調整
- **復古模式**: 米色背景，減少疲勞

### OCR 設定
- **快速模型**: 快速識別，適合一般用途
- **最佳模型**: 高精度識別，適合重要文件

### 功能設定
- **自動儲存**: 自動儲存閱讀進度
- **記住最後檔案**: 啟動時自動載入上次檔案
- **觸控導航**: 支援觸控螢幕操作
- **顯示頁碼**: 顯示當前頁碼

## 🔧 故障排除

### 常見問題

#### 1. 應用程式無法啟動
**問題**: 啟動時出現錯誤
**解決方案**:
- 確認 Java 版本為 17 或更高
- 確認 JavaFX 已正確安裝
- 檢查系統記憶體是否充足

#### 2. OCR 功能無法使用
**問題**: 文字識別失敗
**解決方案**:
- 確認 `tessdata` 資料夾位置正確
- 檢查語言檔案是否完整
- 嘗試切換 OCR 模型

#### 3. 檔案載入失敗
**問題**: 無法開啟特定檔案
**解決方案**:
- 確認檔案格式是否支援
- 檢查檔案是否損壞
- 嘗試重新啟動應用程式

#### 4. 亮度調整異常
**問題**: 亮度調整效果不正確
**解決方案**:
- 使用預覽功能確認效果
- 點擊「應用設定」按鈕
- 重新啟動應用程式

## 📊 效能優化
